Ordered List of HTML Tags
  1. <html>: The root element that encloses the entire HTML document.
  2. <head>: Contains meta-information and resources for the HTML document.
  3. <title>: Sets the title of the HTML document, which appears in the browser's title bar or tab.
  4. <meta>: Provides metadata about the HTML document, such as character encoding and viewport settings.
  5. <link>: Establishes relationships between the current document and external resources, such as stylesheets.
  6. <style>: Contains embedded CSS for defining the page's appearance.
  7. <script>: Embeds or links to JavaScript for adding interactivity to the web page.
  8. <body>: Contains the visible content of the HTML document.
  9. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Headings that indicate the structure and hierarchy of content.
  10. <p>: Represents a paragraph of text.
  11. <a>: Defines hyperlinks, allowing users to navigate to other web pages or resources.
  12. <img>: Embeds images in the web page.
  13. <ul>: Creates an unordered (bulleted) list.
  14. <ol>: Creates an ordered (numbered) list.
  15. <li>: Represents a list item within <ul> or <ol> elements.
  16. <div>: A block-level container for grouping and styling content.
  17. <span>: An inline container for applying styles or scripting to small portions of text.
  18. <table>: Defines a table for organizing data in rows and columns.
  19. <form>: Encloses form elements for user input.
  20. <input>: Provides various types of form input fields.
  21. <button>: Creates a clickable button in a form.
  22. <textarea>: Defines a multi-line text input field within a form.
  23. <select>: Generates a dropdown selection list within a form.
  24. <iframe>: Embeds an external web page within the current page.
  25. <hr>: Inserts a horizontal line for visual separation within the page.
  26. <br>: Inserts a line break within text or content.
  27. <pre>: Defines preformatted text, preserving whitespace and line breaks.
  28. <blockquote>: Indicates a block of quoted text.
